【けものフレンズ 第3話より トキ】

このデータを扱うにはMikuMikuDance（以下MMDと表記）Ver.7.39.以降（必須）と
MMEffect ver0.26（以下MMEと表記）が必要です

MMD本体の入手先
http://www.geocities.jp/higuchuu4/
MMEの入手先
http://www.nicovideo.jp/watch/sm12149815


■■Modelフォルダの主な内容■■
・トキ.pmx	…キャラクターモデル
・toki.png	…メインテクスチャ
・toki2.png	…襟巻用の単色テクスチャ
・toki3.png	…まゆ用の単色テクスチャ
・Eyebrows.x	…まゆ用のポストエフェクトパネルファイル
・Eyebrows.fx	…まゆ用のポストエフェクトファイル
・DrowEyebrows_toki.fx	…まゆ用のポストエフェクトの設定ファイル
